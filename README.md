# Recipe Tutorial Website

A modern web application for sharing and discovering cooking recipes, built with Vue 3 and AWS services.

## Features

- Browse and search recipes
- Detailed recipe views with ingredients and instructions
- User authentication and personalized recipe management
- Recipe ratings and reviews system
- Responsive design for all devices
- Image upload and management for recipes

## Tech Stack

### Frontend
- Vue 3 with Vue Router
- TypeScript
- PrimeVue UI Components
- Tailwind CSS
- Font Awesome Icons

### Backend/Infrastructure
- AWS Amplify
- AWS Storage
- AWS Authentication
- AWS DataStore

## Dependencies

### Core Dependencies
- Vue 3.4.21
- Vue Router 4.4.5
- AWS Amplify 6.4.0
- PrimeVue 4.2.1
- Tailwind CSS 3.4.14

### Development Tools
- Vite 5.2.8
- TypeScript 5.4.5
- AWS CDK 2.137.0
- PostCSS 8.4.47

## Development

To get started with development:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
